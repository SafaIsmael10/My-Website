MedKit: Medical Appointment Booking Software

This was my final year project in university. Made using PHP, HTML, CSS and JavaScript.

This is a medical clinic appointment booking website which has patient and doctor sides depending on logins. Bookings go to a database and are securely stored and seen by doctors for their schedule.

The unique feature for this was a body diagram with different information on illnesses/injuries depending on which body part is clicked, this way patients can have an easier time understanding what is wrong with them, and focus on other things in the appointment, ultimately saving time.

There are also other features made to try save doctors time such as referral letter drafts.

Although not 100% complete and not being hosted anymore, it was the gateway to web development and looking into medical software development.
